# Camino a La Crypta 🍻⚡

Una novela visual interactiva de anime japonés que narra la transformación de un "shitcoiner" arruinado en un bitcoiner soberano. 

Acompaña a Nico, un joven trader atrapado en el frenesí del apalancamiento y el FOMO del casino digital, mientras choca de frente contra la realidad de un mercado manipulado. A lo largo de la historia, Nico deberá descender por la madriguera del conejo de Bitcoin, comprender los pilares de la escasez digital, dominar los micropagos instantáneos a través de la red Lightning y buscar el santuario secreto de desarrolladores: **La Crypta**.

---

## 🎮 Características del Juego

- **Narrativa de Decisiones:** Cada elección que tomes tiene peso. Elegir el camino del FOMO o la centralización te llevará a finales alternativos trágicos (Bad Endings), obligándote a replantear tu estrategia y reiniciar el capítulo.
- **Estética Cyberpunk/Anime:** Interfaces y escenas diseñadas en un estilo visual limpio e inmersivo de anime japonés moderno con transiciones dinámicas.
- **Minijuegos Interactivos:**
  - **Exchange Simulator:** Vive la adrenalina y la desesperación de ver colapsar el precio de una memecoin apalancada en tiempo real.
  - **Lightning Wallet Simulator:** Escanea y realiza micropagos ficticios por Lightning Network de forma instantánea, experimentando la velocidad de las comisiones ultra bajas.
- **Audio Procedural Sintetizado:** Música lo-fi de fondo y efectos de sonido retro generados en tiempo real directamente desde tu navegador a través de la API Web Audio de HTML5.

## 🛠️ Instalación y Uso Local

Para correr este proyecto en tu entorno local, asegúrate de tener instalado [Node.js](https://nodejs.org/).

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/CapScabio/Camino.Crypta.git
   cd camino-a-la-crypta
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar el servidor de desarrollo:**
   ```bash
   npm run dev
   ```

4. **Compilar para producción:**
   ```bash
   npm run build
   ```
